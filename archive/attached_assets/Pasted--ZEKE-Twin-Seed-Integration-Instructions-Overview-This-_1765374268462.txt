## Integration Steps for New ZEKE Instance

### 1. Database Initialization
Create your SQLite database with the same schema, then import the JSONL files:

```javascript
// Example: Import memories
const memories = fs.readFileSync('twin_seed/memories/all_memories.jsonl', 'utf8')
  .split('\n')
  .filter(Boolean)
  .map(JSON.parse);

for (const memory of memories) {
  await db.run(`INSERT INTO memory_notes (...) VALUES (...)`, memory);
}
```

### 2. Profile Files
Copy the profile files to the appropriate locations:
- `zeke_profile.md` → Project root
- `zeke_knowledge.md` → Project root
- `replit.md` → Project root (adapt as needed)

### 3. Embeddings
The memories include pre-computed embeddings. If using a different embedding model, you may need to regenerate them:
```javascript
for (const memory of memories) {
  if (!memory.embedding || useNewModel) {
    memory.embedding = await generateEmbedding(memory.content);
  }
}
```

### 4. Contact Permissions
Contacts include access level configurations. Review and apply as needed for the new system.

### 5. Knowledge Graph
Import entities first, then entity_links (they reference entity IDs).

## Data Formats

### JSONL Files
Each line is a valid JSON object. Parse line by line:
```javascript
const data = fs.readFileSync(file, 'utf8')
  .split('\n')
  .filter(line => line.trim())
  .map(JSON.parse);
```

### Embeddings
Stored as JSON string arrays of floats (OpenAI text-embedding-3-small format, 1536 dimensions).
