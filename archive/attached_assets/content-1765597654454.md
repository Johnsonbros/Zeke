## Required Runtimes

[Permalink: Required Runtimes](https://github.com/Johnsonbros/zeke#required-runtimes)

Versions are pinned in `.tool-versions`:

- **Node.js**: 20.11.x
- **Python**: 3.11.x

The `package.json` enforces `engines.node: ">=20.11.0 <21.0.0"`.

## Local Setup

[Permalink: Local Setup](https://github.com/Johnsonbros/zeke#local-setup)

### Install Dependencies

[Permalink: Install Dependencies](https://github.com/Johnsonbros/zeke#install-dependencies)

```
# Node.js dependencies
npm install

# Python dependencies (uses pyproject.toml)
pip install .
```

### Environment Variables

[Permalink: Environment Variables](https://github.com/Johnsonbros/zeke#environment-variables)

Create a `.env` file at the project root. Use `.env.schema` as reference.

**Required variables:**

- `APP_NAME` \- Application name
- `APP_ENV` \- `development`, `staging`, or `production`
- `PORT` \- Server port (default: 5000)
- `DATABASE_URL` \- PostgreSQL connection string
- `JWT_SECRET` \- JWT signing key
- `OPENAI_API_KEY` \- OpenAI API key
- `LOG_LEVEL` \- `debug`, `info`, `warn`, or `error`

### Database Setup

[Permalink: Database Setup](https://github.com/Johnsonbros/zeke#database-setup)

```
npm run db:init    # Initialize SQLite database
npm run db:push    # Apply Drizzle migrations
```

## Running the Dev Server

[Permalink: Running the Dev Server](https://github.com/Johnsonbros/zeke#running-the-dev-server)

```
npm run dev
```

Starts both Express backend and Vite frontend on port 5000.

For Python agents (optional):

```
cd python_agents && uvicorn main:app --reload --port 5001
```

## Deployment

[Permalink: Deployment](https://github.com/Johnsonbros/zeke#deployment)

The project is deployed via Replit. There is no manual deployment process.

- CI runs on push/PR via GitHub Actions (`.github/workflows/ci.yml`)
- Production deployments are managed through Replit's deployment system
- No deployment scripts are included in this repository
