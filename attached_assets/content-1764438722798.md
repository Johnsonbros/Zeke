## üöÄ Getting Started

[Permalink: üöÄ Getting Started](https://github.com/limitless-ai-inc/limitless-api-examples#-getting-started)

To use these examples, you'll need:

- [An API key](https://limitless.ai/developers) from Limitless
- [Python 3](https://realpython.com/installing-python/) and virtualenv (you can install virtualenv with `pipx install virtualenv`)
- Basic understanding of REST APIs

You can also use our [OpenAPI spec](https://github.com/limitless-ai-inc/limitless-api-examples/blob/main/openapi.yml) to generate client libraries in your language of choice.

### Export Markdown

[Permalink: Export Markdown](https://github.com/limitless-ai-inc/limitless-api-examples#export-markdown)

This example simply prints the Markdown content from each the most recent record.

```
LIMITLESS_API_KEY="your_api_key" python export_markdown.py
```

### Generate a daily summary from your transcripts

[Permalink: Generate a daily summary from your transcripts](https://github.com/limitless-ai-inc/limitless-api-examples#generate-a-daily-summary-from-your-transcripts)

```

```
