## Required Runtimes

The `package.json` enforces `engines.node: ">=20.11.0 <21.0.0"`.

[Permalink: Local Setup](https://github.com/Johnsonbros/Zeke/tree/main#local-setup)

### Install Dependencies

[Permalink: Install Dependencies](https://github.com/Johnsonbros/Zeke/tree/main#install-dependencies)

```
# Node.js dependencies
npm install

# Python dependencies (uses pyproject.toml)
pip install .
```

### Environment Variables

[Permalink: Environment Variables](https://github.com/Johnsonbros/Zeke/tree/main#environment-variables)

Create a `.env` file at the project root. Use `.env.schema` as reference.

**Required variables:**

- `APP_NAME` \- Application name
- `APP_ENV` \- `development`, `staging`, or `production`
- `PORT` \- Server port (default: 5000)
- `DATABASE_URL` \- PostgreSQL connection string
- `JWT_SECRET` \- JWT signing key
- `OPENAI_API_KEY` \- OpenAI API key
- `LOG_LEVEL` \- `debug`, `info`, `warn`, or `error`

### Database Setup

[Permalink: Database Setup](https://github.com/Johnsonbros/Zeke/tree/main#database-setup)

```
npm run db:init    # Initialize SQLite database
npm run db:push    # Apply Drizzle migrations
```

## Running the Dev Server

[Permalink: Running the Dev Server](https://github.com/Johnsonbros/Zeke/tree/main#running-the-dev-server)

```
npm run dev
```

Starts both Express backend and Vite frontend on port 5000.

For Python agents (optional):

```
cd python_agents && uvicorn main:app --reload --port 5001
```
